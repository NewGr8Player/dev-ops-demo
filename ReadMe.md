# dev-ops-demo

> 主要用来展示Dev-ops的基本功能与使用
>
> 初始化SpringBoot项目地址：`https://start.spring.io/`

## 常规的解决方案 

| 功能     | 组件或方案                                                      |
|--------|------------------------------------------------------------|
| 版本控制   | GitLab/Gitea                                               |
| 镜像仓库   | Harbor/私服nexus [@See DockerRegistry.md](DockerRegistry.md) |
| 容器     | Docker                                                     |
| 编排     | K8S                                                        |
| 服务治理   | Consul                                                     |
| 脚本语言   | Python                                                     |
| 日志管理   | Cat+Sentry/ELK                                             |
| 系统监控   | Prometheus                                                 |
| 负载均衡   | Nginx                                                      |
| 网关     | Kong && Zuul                                               |
| 链路追踪   | Zipkin                                                     |
| 产品和UI图 | [蓝湖](https://lanhuapp.com/)                                |
| 内部文档   | Confluence                                                 |
| 报警     | 推送到工作群,通过Api推送到钉钉、微信等                                      |
